              전략 실행 전 데이터 검증

            어떻게 쓰나백테스트 전에 데이터가 충분한지, 극단적 변동성 종목을 제외해야 하는지 판단합니다.
            
              
            결과 활용점수가 높아도 매수 신호가 아니라 후보군 품질 점검용입니다. 실행 판단은 시그널 매트릭스와 백테스트에서 확인합니다.
            
            
            
            
            
              
            
            
            
            
              
            백테스트 종목과 기간을 먼저 정한 뒤 실행하면, 어떤 종목이 후보로 남을 만한지와 제외해야 할 데이터 문제를 표시합니다.
            
            
            
          
            
          
            
            
            
              
            시그널 매트릭스
            
              다음 봉 체결 기준
            
            
            
            
              
            후보 선정백테스트 종목·기간·전략·Top N 조건을 그대로 가져와 현재 시점의 후보와 비후보를 나눕니다.
            
              
            판정 근거각 종목의 점수, 사용된 팩터, 제외 사유, 데이터 신선도 경고를 함께 보여줍니다.
            
              
            체결 기준신호 발생일과 다음 봉 체결일을 분리해 룩어헤드 없이 실행 가능한지 확인합니다.
            
            
            
            
            
              
            
            
            
            
              
            시그널을 생성하면 매수 후보, 보류 종목, 제외 사유, 다음 봉 체결일, 데이터 경고를 표로 확인합니다.
